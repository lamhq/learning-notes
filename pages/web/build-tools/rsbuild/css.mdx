# CSS

## Import CSS

Import CSS in `node_modules`.

Import in a JS file:

```js filename="src/index.js"
/* reference normalize.css */
/* https://github.com/necolas/normalize.css */
import 'normalize.css';
```

Import in a CSS file::
```json
@import 'normalize.css';

body {
  /* */
}
```


## Use Tailwind CSS

You can integrate Tailwind CSS in Rsbuild via PostCSS plugins.

Register the `tailwindcss` PostCSS plugin through `postcss.config.cjs` or `tools.postcss`.

```js filename="postcss.config.cjs"
module.exports = {
  plugins: {
    tailwindcss: {},
  },
};
```

Then, install Tailwind CSS and create a `tailwind.config.js` file in the root directory:
```shell npm2yarn
npm add tailwindcss -D
```

```js filename="tailwind.config.js"
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['./src/**/*.{html,js,ts,jsx,tsx}'],
  theme: {
    extend: {},
  },
  plugins: [],
};
```
