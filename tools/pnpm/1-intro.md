# Introduction

## Motivation

- **Saving disk space**.
- **Boosting installation speed**.
- **non-flat `node_modules`**.


## Installation

Install using `npm`:

```shell
npm install -g pnpm
```


### Using a standalone script (POSIX)

```bash
curl -fsSL https://get.pnpm.io/install.sh | sh -

# or
wget -qO- https://get.pnpm.io/install.sh | sh -
```
