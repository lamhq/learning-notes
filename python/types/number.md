## Numbers

Available operators:

- `+`, `-`, `*`, `/`
- `//`, `%`, `**`

```python
# classic division returns a float
17 / 3  # 5.666666666666667

# floor division discards the fractional part
17 // 3 # 5

# the % operator returns the remainder of the division
17 % 3  # 2
2

# 5 squared
5 ** 2  # 25
```
