## Use VIM to fix git merge problem
GIT PROBLEM / ERROR : ‘PLEASE ENTER A COMMIT MESSAGE TO EXPLAIN WHY THIS MERGE IS NECESSARY …’

1. press "i"
2. write your merge message
3. press "esc"
4. write ":wq"
5. then press enter
