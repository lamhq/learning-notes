# Data Analyst Roadmap

## Required skills

- Advanced Microsoft Excel
- SQL
- Python (NUMPY, PANDAS, MATPLOTLIB, ...) or R
- Data visualization tools: Tableau
- PowerBI
- Statistical Skills
- Mathematical Skills
