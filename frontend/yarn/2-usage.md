# Usage

## Starting a new project

```bash
yarn init
```

## Installing all the dependencies

```bash
yarn
yarn install
```

## Adding a dependency

```bash
yarn add [package]
yarn add [package]@[version]
yarn add [package]@[tag]

yarn add [package] --dev
yarn add [package] --peer
```

## Removing a dependency

```bash
yarn remove [package]
```

## Upgrading a dependency

```bash
yarn up [package]
```
